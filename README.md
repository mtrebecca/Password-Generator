# Gerador de Senhas

Este é um aplicativo web simples para gerar senhas aleatórias com características personalizáveis, como letras minúsculas, letras maiúsculas, números e símbolos. O aplicativo também fornece um indicador visual da força da senha com base nas opções selecionadas.

## Funcionalidades

- Gerar senhas com opções personalizáveis (letras minúsculas, letras maiúsculas, números, símbolos).
- Feedback visual sobre a força da senha.
- Capacidade de copiar a senha gerada para a área de transferência.
- Design responsivo e interface amigável.

## Tecnologias Utilizadas
- HTML5: Estrutura da aplicação.
- CSS3: Estilos, utilizando SCSS para melhor gerenciamento.
- JavaScript (ES6+): Lógica para geração de senhas e interações do usuário.
- Webpack: Ferramenta de empacotamento de módulos para compilar arquivos JavaScript.
- Sass: Pré-processador CSS para estilização.
- lite-server: Servidor de desenvolvimento leve com recarregamento ao vivo.

### Iniciar Projeto

1. Clone o Repositório

`git clone https://github.com/mtrebecca/Password-Generator.git`

2. Instale as Dependências

`npm install`

3. Executar o Projeto

`npm start`

4. Acessar o Projeto

`http://localhost:3000`
